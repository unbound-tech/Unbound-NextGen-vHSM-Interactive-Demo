UKC Admin

UKC has a web administration portal. You can log into this portal as either the Root SO or the Partition SO.

Note: To make this demo faster to deploy, the URLs are accessed over HTTP. In a production system, they would only be used with HTTPS.

Root SO ?

The Root Security Officer (SO) manages UKC partitions.

A UKC partition namespaces UKC into segregated domains that specify key material, authorized users, and certified client machines.

Log into UKC Admin as Root SO

Partition SO ?

The Partition SO manages a single partition, including configuring access permissions and creating secure objects (such as keys and secrets).

Log into UKC Admin as Partition SO
